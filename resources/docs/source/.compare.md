## Login Action
return token and some details for user authentication
[
&#039;response&#039; =&gt; &#039;success&#039;,
&#039;access_token&#039; =&gt; $token,//accessToken
 &#039;token_type&#039; =&gt; &#039;bearer&#039;,//TypeOfToken it will be type of authentication Bearer
 &#039;expires_in&#039; =&gt; config(&#039;jwt&#039;)[&#039;ttl&#039;] * 60 //time in sec token will expired in
]

> Example request:

```bash
curl -X POST "http://localhost/api/auth/login" \
-H "Accept: application/json" \
    -d "email"="mdibbert@example.org" \
    -d "password"="eveniet" \

```

```javascript
var settings = {
    "async": true,
    "crossDomain": true,
    "url": "http://localhost/api/auth/login",
    "method": "POST",
    "data": {
        "email": "mdibbert@example.org",
        "password": "eveniet"
},
    "headers": {
        "accept": "application/json"
    }
}

$.ajax(settings).done(function (response) {
    console.log(response);
});
```
